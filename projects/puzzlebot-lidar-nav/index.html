The problem

A mobile robot navigating a maze with only a 2D LiDAR has to tell the difference between "the wall just ended" and "this is an outer corner" — two situations that look nearly identical in raw scan data but demand opposite navigation decisions.

Approach

Combined Bug 0 and Bug 2 wall-following strategies with an EKF for pose estimation and SLAM for mapping, plus ArUco markers for waypoint correction. The key contribution was a jump-based discontinuity detector on the LiDAR range data (OUTER_ESCAPE_JUMP_THRESHOLD = 0.50 m) that reliably distinguishes outer corners from wall ends in real time.

  • Bug 0 / Bug 2 hybrid wall-following logic
  • EKF localization fused with SLAM-built maps
  • ArUco marker detection for waypoint correction
  • Custom LiDAR jump-detection for corner ambiguity

Results

The robot completed 5-waypoint maze navigation with an average real-world error of roughly 5 cm, in collaboration with Manchester Robotics UK.
