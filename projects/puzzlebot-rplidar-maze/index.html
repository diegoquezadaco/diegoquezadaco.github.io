Autonomous Maze Navigation — Puzzlebot RPLIDAR Edition

Autonomous exploration challenge developed in collaboration with Manchester Robotics (UK).

Start Date March 2026
End Date June 2026
Duration ~4 months
Role Lead Developer & Project Manager
Technologies ROS2, RPLIDAR A1, EKF, ArUco

Overview

An autonomous exploration challenge developed in collaboration with Manchester Robotics (UK), tasking a differential-drive Puzzlebot with navigating an unknown maze using only onboard LiDAR, wheel encoders, and ArUco marker detections — no external localization.

Responsibilities

  • Led a team of four to design and build a full autonomous navigation stack on a differential-drive robot.
  • Implemented Bug 0 and Bug 2 navigation algorithms together with a wall-following controller.
  • Developed a custom LiDAR discontinuity detector to overcome RPLIDAR A1 hardware limitations.
  • Fused wheel encoder odometry with an Extended Kalman Filter and ArUco marker detections to reduce localization drift.
  • Managed team planning, task delegation, and integration testing across the four-month development cycle.

Technical Challenges

  • Disambiguating outer maze corners from wall endings using only 2D LiDAR range data — solved with a custom jump-based discontinuity detector (OUTER_ESCAPE_JUMP_THRESHOLD = 0.50 m) that flags large radial jumps between consecutive LiDAR readings to distinguish a genuine outer corner from open space where a wall simply ends.
  • Reducing cumulative odometry drift over long maze traversals by fusing encoder data with sparse ArUco marker detections in an EKF.
  • Tuning the wall-following controller to reliably handle inner corners, outer corners, and wall endings without oscillation or stalling.
  • Coordinating a four-person team across perception, control, localization, and integration workstreams.

Results

  • Achieved five-waypoint maze navigation with approximately 5 cm average real-world localization error.
  • Successfully demonstrated robust handling of inner corners, outer corners, and wall-ending ambiguity in live trials.
