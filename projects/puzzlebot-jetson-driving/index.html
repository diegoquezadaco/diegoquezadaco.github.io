Overview

A fully vision-based autonomous driving challenge built on the Jetson-powered Puzzlebot platform, developed in collaboration with Manchester Robotics (UK). The system combines classical computer vision for lane following with a custom-trained deep learning model for traffic sign and signal recognition, integrated into a single closed-loop ROS2 driving pipeline.

Responsibilities

  • Developed a complete ROS2 autonomous driving pipeline based entirely on computer vision.
  • Implemented a robust lane-following algorithm using classical image processing techniques.
  • Trained a custom YOLOv8n model for traffic sign and traffic light detection.
  • Integrated perception and control modules into a single fully autonomous driving system running on Jetson hardware.

Technical Challenges

  • Running real-time perception (lane detection + YOLOv8n inference) within the compute and power constraints of the Jetson platform.
  • Making lane detection robust to lighting variation and track imperfections using only classical CV, without relying on deep learning for that stage.
  • Cleanly integrating a learned perception module (YOLOv8n) with a rule-based control stack so detections translated reliably into driving decisions.

Results

  • Delivered a fully autonomous vision-only driving system, from perception through control, running live on Jetson hardware.
  • Successfully demonstrated reliable lane following combined with correct traffic sign and light responses in test runs.
