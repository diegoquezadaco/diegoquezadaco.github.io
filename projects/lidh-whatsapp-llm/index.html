The problem

Users needed a conversational way to interact with the product directly inside WhatsApp, with responses reliable enough to handle real, unscripted traffic.

Approach

Built the conversational pipeline end to end: message handling, conversational state management, prompt design, and guardrails around response reliability, deployed against the WhatsApp Business API.

  • Conversational state tracked across multi-turn exchanges
  • Prompt design tuned for consistent, on-brand responses
  • Deployed and monitored against real user traffic

Results

Shipped a working assistant handling real conversations in production for Lidh's user base.
